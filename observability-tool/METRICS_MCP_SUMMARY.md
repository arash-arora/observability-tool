# Metric Scores & MCP Navigation Update

## 1. Evaluations Detail Scores
- Updated the "View Details" modal in the **Evaluations ** page.
- Now displays a **Metric Scores** card breakdown for each evaluation.
- Dynamically shows relevant metrics based on evaluation type:
  - **RAG**: Faithfulness, Answer Relevance, Context Precision/Recall.
  - **Agent**: Tool Selection, Input Structure, Error Recovery.
  - **Workflow**: Accuracy, Latency Score, Output Format.

## 2. MCP Integration Navigation
- Added **"MCP & Tools"** to the main sidebar navigation.
- This provides direct access to the new MCP page where **ServiceNow**, **Slack**, and **Jira** integrations are now located (under "External Tools").
